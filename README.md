# Purchase Orders App (Vite + React + MirageJS)

Mobile-first scaffold for a purchase order workflow with Manager, Admin, and Buyer roles.

## Quick start

1. `npm install`
2. `npm run dev`
3. Open http://localhost:5173

MirageJS provides an in-browser fake API under `/api/*` in development. To connect a real backend change `src/config.js`.
